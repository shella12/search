# 📖 Weather App
 <a name="about-project"></a>

> This is a page allowing users to search for names, displaying a list of matching individuals.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>
<details>
  <summary>Client</summary>
  <ul>
    <li><a href="https://reactjs.org/">React.js</a></li>
  </ul>
</details>
<details>
  <summary>Server</summary>
  <ul>
    <li><a href="https://cloud.google.com/">Ruby on Rails</a></li>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
  </ul>
</details>

### Key Features <a name="key-features"></a>

- Search: Allows users to search for names, displaying a list of matching individuals
- Add to favourite:  Provides an option for users to save or favorite individuals 
- Show recent search: Displays a list showcasing up to 10 of the most recent search queries.
- Rest API: The app uses Rest API to get data from server.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Node.js
- Git
- Code Editor
- Browser

### Setup

Clone this repository to your desired folder:

```sh
  -cd my-folder
  -git clone https://github.com/shella12/search.git
```

## Install

```sh
 npm install --save -dev
```

## Start server

``` npm start ```

## To test the app run following command

``` npm test ```
